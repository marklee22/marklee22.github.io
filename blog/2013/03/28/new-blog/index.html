New Blog!

Given my new web skills, I decided to switch blogs! Although, Wordpress was a great way to start my blogging career, I wanted something a little more customizable to allow me to include some of the projects I’ve been working on lately. I switched to Octopress, which is based on the Jekyll blog static-page framework.

I’m now using github to host my blog which also allows free hosting. I’ve only just started playing around with what Octopress has to offer, but so far I’m impressed how easy it was to setup. Overall, it probably took me 3-4 hours to get everything setup and migrated. I guess the one drawback is I lost the nifty spell checker before I post, so maybe if I’m really bored, I’ll work on a javascript or Ruby plugin to catch any errors in the future.
